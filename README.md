🌐 Portfolio Website – Learning Project (Next.js)
תיאור הפרויקט

זהו פרויקט פורטפוליו אישי לימודי, שנבנה מחדש באמצעות Next.js (App Router) כחלק מתהליך למידה והדגמה עבור עסק חוגי תכנות ובניית אתרים.

הפרויקט מדגים איך אתר React קלאסי יכול להפוך לאתר מודרני, מהיר ומותאם לסטנדרטים עדכניים – בדגש על ארכיטקטורה נכונה, חוויית משתמש, וניהול קוד מקצועי.
